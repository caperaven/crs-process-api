# class - DomObserverActionsUse this class for observing dom changes e.g. ResizeObserver @description Use this class for observing dom changes e.g. ResizeObserver    ## Features - observe_resize - observer resize for an element   - unobserve_resize - unobserve resize for an element  ## method - observe_resizeObserve the element for changes in size.  When the size changes, the callback is called.***parameters***|parameter|type|description|required|default||---------|----|-----------|--------|-------||step|Object|The step to perform|true|-||context|Object|The context of the process|true|-||process|Object|The process|true|-||item|Object|The item being processed|true|-||step.args.element|HTMLElement|The element to observe|true|-||step.args.callback|Function|The callback to call when the size changes|true|-|**Returns**: {Promise<void>}  ***Example: javascript example***```js await crs.call("dom_observer", "observe_resize", {      element: element,      callback: callback   }, context, process, item);    ```***Example: json example***```json {       "type": "dom_observer",       "action": "observe_resize",       "args": {           "element": "element",           "callback": "callback"        }   }  ```## method - unobserve_resizeDisable the resize observer for the element***parameters***|parameter|type|description|required|default||---------|----|-----------|--------|-------||step|Object|The step to perform|true|-||context|Object|The context of the process|true|-||process|Object|The process|true|-||item|Object|The item being processed|true|-||step.args.element|HTMLElement|The element to unobserve|true|-|**Returns**: {Promise<void>}  ***Example: javascript example***```js  await crs.call("dom_observer", "unobserve_resize", {        element: element    }, context, process, item);    ```***Example: json example***```json  {        "type": "dom_observer",        "action": "unobserve_resize",        "args": {            "element": "element"        }      }  ```
