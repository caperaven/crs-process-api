# class - DomUtilsActionsThis contains functions for utility  ## Features -call_on_element - call a function on a element   -get_property - get a element's property value   -set_properties - set a element's property value   -open_tab - open a new tab   -get_element_bounds - get the bounds of an element   -find_parent_of_type - find the parent of a given type   - #find_parent_of_type - find the parent of a given type  ## method - call_on_elementCall a function on a element  ***parameters***|parameter|type|description|required|default||---------|----|-----------|--------|-------||step|Object|The step object.|true|-||context|Object|The context of the current process.|true|-||process|Object|The current process|true|-||item|Object|The item that is being processed.|true|-||step.args.element|String|The id of the element to call the function on.|true|-||step.args.function|String|The name of the function to call.|true|-||step.args.args|Array|The arguments to pass to the function.|true|-||step.args.target|String|The target to store the result in.|true|-|***Example: javascript***```js await crs.call("dom_utils", "call_on_element", {     element: "my-element",     action: "myFunction",     args: ["my", "arguments"]   });    ```***Example: json***```json {    "type": "dom_utils",    "action": "call_on_element",    "args": {      "element": "my-element",      "action": "myFunction",      "args": ["my", "arguments"]     }   }    ```**Returns**: {Promise<>}## method - get_propertyGet a element's property value***parameters***|parameter|type|description|required|default||---------|----|-----------|--------|-------||step|Object|The step object.|true|-||context|Object|The context of the current process.|true|-||process|Object|The current process|true|-||item|Object|The item that is being processed.|true|-||step.args.element|String|The id of the element to call the function on.|true|-||step.args.property|String|The name of the property to get.|true|-||step.args.target|String|The target to store the result in.|false|-|***Example: javascript***```js await crs.call("dom_utils", "get_property", {     element: "my-element",     property: "myProperty"     target: "my-target"    });    ```***Example: json***```json {    "type": "dom_utils",    "action": "find_parent_of_type",    "args": {      "element": "my-element",      "nodeName": "DIV",      "target": "my-target"     }   }  ```**Returns**: {Promise<Element|undefined>} ToDo: AW - Ask about this method use-case for example.  ## method - #findParentOfTypeRecursively finds a parent or ancestor that matches a specific tagName, optionally stopping at a specified node for performance.    ***parameters***|parameter|type|description|required|default||---------|----|-----------|--------|-------||element|Element|The element to start searching from.|true|-||nodeName|string|The name of the node to find.|true|-||stopAtNodeName|string|The name of the node to stop at.|true|-||stopAtNodeQuery|string|The query to find the node to stop at.|true|-||nodeQuery|string|The query to find the node.|true|-|***Example: javascript***```js await crs.call("dom_utils", "find_parent_of_type", {     element: "my-element",     nodeName: "DIV"   });
