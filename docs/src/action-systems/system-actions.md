# class - SystemActionsThe system actions are used to perform system functions. @description This class contains functions that are used to perform system actions.    ## Features perform - The perform function is used to call the action function.   copy_to_clipboard - Copy the value of the source to the clipboard.   sleep - Sleep function used for testing to emulate network traffic delay.   pause - Pause the process for the specified duration.   resume - Resume the process after a pause.   abort - Abort the process.   ismobile - Check if the current device is a mobile device.   is_portrait - Check if the current device is in portrait mode.   is_landscape - Check if the current device is in landscape mode.  ## method - Copy the value of the source to the clipboard***parameters***|parameter|type|description|required|default||---------|----|-----------|--------|-------||step|-|The step object from the process definition.|true|-||context|-|The context object that is passed to the process.|true|-||process|-|the process object|true|-||item|-|The current item being processed.|true|-||step.args.source|string|The source value to copy to the clipboard.|true|-|**Returns**: {Promise<void>}  ***Example: javascript example***```js await crs.call("system", "copy_to_clipboard", {       source: "$context.value"   }, context, process, item);    ```***Example: json example***```json {       "action": "copy_to_clipboard",       "args": {           "source": "$context.value"        }   }  ```## method - Sleep function used for testing to emulate network traffic delay @description "Wait for the specified duration before continuing."  ***parameters***|parameter|type|description|required|default||---------|----|-----------|--------|-------||step|-|The step object from the process definition|true|-||context|-|The context object that is passed to the process.|true|-||process|-|The process object|true|-||item|-|The item that is being processed.|true|-||step.args.duration|integer|The duration to wait in milliseconds.|true|-||step.args.target|string|The target to copy the value to.|true|-|**Returns**: A promise that resolves after the duration has passed.  ***Example: javascript example***```js await crs.call("system", "sleep", {      duration: 1000   }, context, process, item);    ```***Example: json example***```json {       "type": "system",       "action": "sleep",       "args": {           "duration": 1000       }   }  ```## method - The function pauses the process and waits for the user to resume the process***parameters***|parameter|type|description|required|default||---------|----|-----------|--------|-------||step|-|the current step in the process.|true|-||context|-|the context of the process.|true|-||process|-|The process object that is being run.|true|-||step.args.bId|string|The binding context id to use for the resume function.|true|-|**Returns**: A promise.  ***Example: javascript example***```js await crs.call("system", "pause", {        bId: "myBindingContext"   }, context, process, item);
